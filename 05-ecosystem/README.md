## Security Considerations

⚠️ **Smart Contract Risks:**
- Contracts are upgradeable (governance controlled)
- Always verify contract addresses
- Check recent audits

⚠️ **Agent Risks:**
- Agent code runs off-chain (trust agent operator)
- Check agent reputation before high-value tasks
- Use validation for important work

⚠️ **Validation Risks:**
- Different trust models have different guarantees
- Validators can be malicious (check their reputation)
- Use multiple validators for critical work
